# FUTURE_CS_02

## Phishing Email Detection & Awareness System

### Objective
Analyze phishing emails and identify malicious indicators such as fake domains, SPF/DKIM failures, suspicious links, and urgent language.

### Tools Used
- Google Admin Toolbox Message Header Analyzer
- MXToolbox
- Google Chrome
- MS Word
- GitHub

### Features
- Email Header Analysis
- Risk Classification
- Phishing Indicator Detection
- Awareness Guidelines
- Prevention Tips

### Risk Classification
PHISHING (High Risk)

### Prevention Guidelines
- Verify sender domains
- Avoid suspicious links
- Do not open unknown attachments
- Enable multi-factor authentication
- Report suspicious emails immediately

### Outcome
This project demonstrates phishing email analysis and cybersecurity awareness practices used in real-world security operations.
